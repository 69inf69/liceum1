# GitHub User Finder

**Автор:** [Серякова Наталья Николаевна]

## Описание

Приложение для поиска пользователей GitHub с возможностью сохранения избранных профилей.

### Функциональность:
* поиск пользователей GitHub по имени;
* отображение основной информации о пользователе;
* добавление пользователей в избранное;
* сохранение избранных пользователей в JSON‑файл;
* графический интерфейс на Tkinter.

## Использование GitHub API

Приложение использует публичный API GitHub без необходимости аутентификации для базового поиска пользователей.

**Базовый URL:** `https://api.github.com/users/{username}`

**Возвращаемые данные:**
* `login` — имя пользователя;
* `name` — полное имя;
* `public_repos` — количество публичных репозиториев;
* другая информация о профиле.

## Инструкция по использованию

1. Запустите приложение: `python app.py`.
2. Введите имя пользователя GitHub в поле поиска.
3. Нажмите «Найти» — отобразится информация о пользователе.
4. Нажмите «Добавить в избранное», чтобы сохранить пользователя.
5. Избранные пользователи отображаются в нижней части окна.
6. Используйте «Удалить из избранного», чтобы убрать пользователя из списка.

## Примеры использования

**Тест 1: успешный поиск**
* Ввод: `octocat` (официальный тестовый аккаунт GitHub).
* Ожидаемый результат: отображается информация о пользователе, кнопка «Добавить в избранное» работает.

**Тест 2: пустой ввод**
* Ввод: оставить поле пустым.
* Ожидаемый результат: сообщение об ошибке «Поле поиска не должно быть пустым!».

**Тест 3: несуществующий пользователь**
* Ввод: `thisusernamedoesnotexist12345`.
* Ожидаемый результат: сообщение «Пользователь не найден».
